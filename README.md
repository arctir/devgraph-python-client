# Devgraph Python Client

Auto-generated Python client for the [Devgraph](https://devgraph.ai) API.

## Installation

```bash
pip install devgraph-client
```

Or with Poetry:

```bash
poetry add devgraph-client
```

## Development

This client is automatically generated from the Devgraph API OpenAPI specification.

### Setup

```bash
poetry install
```

### Running Tests

```bash
poetry run pytest
```

### Linting

```bash
poetry run ruff check src/
poetry run mypy src/
```

## Versioning

This client follows semantic versioning and is automatically updated when the Devgraph API specification changes.

- **API Version**: See [API_VERSION](https://github.com/arctir/devgraph-api/blob/main/API_VERSION)
- **Client Version**: See `pyproject.toml`
